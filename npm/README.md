# @kshitizz36/provn

Installs the `provn` CLI by downloading the correct pre-built binary from GitHub Releases during `npm install`.

## Install

```bash
npm install -g @kshitizz36/provn
```

## Notes

- Requires Node.js 18+
- Downloads platform-specific binaries from `kshitizz36/Provn` releases
- If no release artifact is available for your platform yet, build from source from the main repository
